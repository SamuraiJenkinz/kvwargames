**Procedure:**

   - Open browser devtools → Network tab → filter on the LLM endpoint (likely `/api/chat` or similar per `src/lib/llmClient.ts`).

   - Copy the full JSON response body.

   - Locate Finch's entry in `responses[]` (`speaker: "finch"`).

   - **PASS criteria:** If `stateUpdate.crisisSeverity` is ≥ 3 AND `stateUpdate.crisisState === "Security-Related Supply Crisis"`, this is a PASS. 6. **If FAIL, stop and escalate.** Do not mark requirements complete.
