    1. Add SPAStaticFiles class to `backend/app/main.py`:
       ```python
       from fastapi.staticfiles import StaticFiles
       from starlette.exceptions import HTTPException as StarletteHTTPException

       class SPAStaticFiles(StaticFiles):
           async def get_response(self, path: str, scope):
               try:
                   return await super().get_response(path, scope)
               except StarletteHTTPException as ex:
                   if ex.status_code == 404:
                       return await super().get_response("index.html", scope)
                   raise ex
       ```

    2. Add conditional SPA mount AFTER all router includes:
       ```python
       import os
       dist_dir = os.path.join(os.path.dirname(__file__), "..", "..", "dist")
       if os.path.isdir(dist_dir):
           app.mount("/", SPAStaticFiles(directory=dist_dir, html=True), name="spa")
       ```
       The dist_dir path resolves from `backend/app/main.py` up to project root `dist/`.
       The `os.path.isdir` check means the static mount is skipped in dev mode (no dist/ until `pnpm build`).

    CRITICAL: The `app.mount("/", ...)` line MUST be the LAST line in the module — after all `app.include_router()` calls. Mount order matters: SPA mount is a catch-all that would swallow API routes if registered first.

    3. Update `.env.example` at project root to fix the naming mismatch and add missing vars:
