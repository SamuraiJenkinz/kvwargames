Card layout details:
- Both cards must visually balance — same min-height (`min-h-[180px]` or similar), same padding, same internal spacing. Use flexbox column with `gap-2` for inner content.

Accessibility:
- Both cards: include an `<h2>` for the title so a screen reader hears "Heading level 2: Load Config" / "Heading level 2: Generate from Brief"
- Disabled card: include an off-screen `<span class="sr-only">Disabled — coming in Phase 7</span>` inside the button for screen readers
- Container has `<main>` semantic wrapper
