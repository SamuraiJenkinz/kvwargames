A facilitation console for policy wargames that actually remembers the conversation.

Three AI advisors respond in character, live game state updates as decisions are made, and every session ends with a downloadable record. Built for the EDIP Council, validated against the production corporate language model under a full five-round scenario.

The premise

Tabletop exercises are the clearest way to pressure-test policy. They are also the hardest to run at pace. A single skilled facilitator has to remember every inject, every card played, every resource the four teams have spent, and deliver consistent voices for each advisory role across a two-hour session.

The Supply Wargame Console does the memory work. The facilitator types what happened; three AI personas respond in character; the crisis tracker, legitimacy score, and per-team resources update in the margin. At the end of the session, a single markdown file captures everything that was said and every state change it caused.

What leadership receives

Every session produces a single downloadable markdown artifact containing session metadata, per-round transcripts with starting-state tables, a debrief section with each advisor's closing reflection, and the final state of play. No screen captures, no manual transcription.

The facilitator remains in control at every step. When an advisor suggests advancing the round or calling the debrief, the console surfaces an explicit confirmation. Nothing is auto-applied. The state tracker shows what changed; the chat feed shows why; the debrief export makes both permanent.

Trust architecture

Corporate language-model credentials never reach the facilitator's browser. Every request passes through an internal proxy that injects the key server-side; the client sees only application payloads. This is verified by network-trace audit on each release.

Malformed responses are caught before they touch game state. A four-layer defensive parser rejects anything that does not conform to the advisor response schema, rendering a clear retry affordance to the facilitator. The session continues without restart, and the incident is captured in the debrief record.

Where it fits

  • Policy Rehearsal
    Stress-test a draft instrument or procedure against a simulated crisis before committing to the real thing.
  • Leadership Induction
    Walk new principals through the pressure points of the instrument portfolio in a single session with a written record.
  • Cross-agency Exercise
    Coordinate four member-state teams through a shared scenario with consistent advisory voices and transparent state.
  • After-action Review
    Revisit past decisions using the debrief record — replay the reasoning, not just the outcome.
